# Task **AirBnB**

AirBnB diagram(use case, activity, class and aequence)



Draw a use-case diagram of adding a new participant and searching for housing by certain parameters on the rental platform 

Draw a class diagram to represent the set of objects necessary to present available apartments, reviews, ratings, photos, owners and reservation options 

Draw an activity diagram of submitting an application for reserving an apartment and receiving (or rejecting) the application by the owner of the apartment

Draw a sequence diagram that shows the submission of the application, its reception and debiting of funds from the user's account to the service's accounts and subsequent transfer of funds to the owner of the apartment

## How to Submit the Lab

To submit your lab work, you need to create a **Pull Request** in this repository. Follow the steps below:

1. **Fork** this repository (click the **Fork** button in the top-right corner of the GitHub page).
2. **Clone** your forked repository to your local machine:
   ```bash
   git clone https://github.com/<your-username>/documenting_lab1.git
   cd documenting_lab1
   ```
3. **Create a new branch** for your submission:
   ```bash
   git checkout -b lab-submission-<your-name>
   ```
4. **Add your diagram files** to the repository folder. Export each diagram as an image (`.png` or `.jpg`) and place the files in the repository directory.
5. **Commit** your changes:
   ```bash
   git add .
   git commit -m "Add lab diagrams - <your-name>"
   ```
6. **Push** your branch to GitHub:
   ```bash
   git push origin lab-submission-<your-name>
   ```
7. **Open a Pull Request**:
   - Go to your forked repository on GitHub.
   - Click **Compare & pull request**.
   - Set the base repository to the original `documenting_lab1` repository and the base branch to `main`.
   - Add a descriptive title (e.g., `Lab submission – <Your Name>`).
   - If your diagrams are images, **attach the image files directly in the Pull Request description** by dragging and dropping them into the text box.
   - Click **Create pull request**.

> **Note:** Even if your diagrams were created in a diagramming tool (e.g., draw.io), you only need to attach the exported image files (`.png`/`.jpg`) to the Pull Request. No special file format is required.
